Fenced code blocks are automatically syntax-highlighted.

## Basic code block

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

Add a language identifier after the opening fence for syntax highlighting:

````mdx
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
````
