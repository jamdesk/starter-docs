## Tabs

Show the same thing across languages or platforms:

<Tabs>
  <Tab title="Node.js">
    ```javascript
    const acme = new Acme('sk_live_abc123');
    const payment = await acme.payments.create({
      amount: 2500,
      currency: 'usd',
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    acme = Acme('sk_live_abc123')
    payment = acme.payments.create(
        amount=2500,
        currency='usd',
    )
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    curl https://api.acme.com/v1/payments \
      -H "Authorization: Bearer sk_live_abc123" \
      -d amount=2500 \
      -d currency=usd
    ```
  </Tab>
</Tabs>
